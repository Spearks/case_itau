# Case Técnico - Itaú

## Visão geral

Este projeto é uma interface interativa desenvolvida para o Case Técnico de Canais e Dados / Comunicação BUPJ. O objetivo da aplicação é atuar como um **Aprovador de Campanhas**, adicionando uma etapa de validação (checklist) antes do disparo de comunicações internas.

## Tecnologias

- Vite
- HTML
- CSS
- React

## Como executar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```
